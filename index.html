Bloomstone PMS — Feature Guide
Quick reference for every section and feature in the system.
⬆ Topbar
Sync dot — green=synced · amber=syncing · red=error
⌕ Search — opens Command Palette (also Ctrl+K)
☼ Daily Brief — today's check-ins, check-outs, auto-status update
＋ Booking — create a new booking
··· Menu — Load/Save Excel · Theme toggle · Sign out
🏠 Dashboard
Stat pills — check-ins, check-outs, active stays, this month revenue, ADR, RevPAR
Contract alerts — red/amber banners if any property contract expires in ≤90 days
Today's Overview — all active stays, arriving & departing guests
Property grid — quick stats per property (occupancy, next booking)
Daily Brief popup — tap ☼ to see status-update actions
📋 Bookings
Click row — open & edit booking
Right-click row (or long-press mobile) — quick status change menu
⎘ Clone — duplicate a booking with same property/guest/rate
⊞ Columns button — show/hide table columns (saved per device)
Checkboxes — select rows → floating bar appears at bottom
Bulk bar — Set Status · Export CSV · Delete for selected rows
Filters — Month · Property · Platform · Status dropdowns
✏️ Booking Drawer (Edit)
Booking tab — guest, property, dates, platform, payment, notes
Financials tab — rate, promo, fees, full breakdown, net revenue
Revenue Split box — appears when property has an owner % set (e.g. 70/30)
History tab — prior bookings for same guest
🖨 Print — printable confirmation with guest details, rates, balance due, house rules
📅 Calendar
Month view — overview of all bookings across properties
Week / Day views — detailed schedule
Timeline view — horizontal Gantt per property
Availability view — free/booked grid by property
Occupancy view — % occupancy heatmap per property
Property pills — filter calendar to one or more properties
💰 Finance
Overview — revenue, expenses, net profit; Owner Payable & Bloomstone Earnings cards if revenue split is set
Expenses — monthly expense tracker per property
Deposits — security deposit collection status
Platforms — manage platform commissions & VAT rates
Owner Statements — select property + month → generate printable payout statement + payout ledger (paid / unpaid history)
🏡 Properties
＋ Add Property — name, city, beds, base rate, extra guest fee, photos, icon
✏ Edit — update any property details
Owner & Contract section — in Edit modal, scroll down → tap "Owner & Contract ▾". Set: owner name/phone/email, revenue share %, payout method, contract start/end dates
🔴 Block Dates — mark dates unavailable for booking
Property card — shows net revenue, occupancy %, owner split badge, contract status
🧑 Guests CRM
Guest cards — auto-built from all bookings. Shows total nights, total revenue, last/next stay
REPEAT badge — appears on guests with 2+ bookings
📋 Guest Prefs — set in booking drawer; visible on guest card and printed confirmation
Click guest — see full booking history for that guest
📈 Reports
Revenue by platform — pie chart breakdown
6-month trend — bar chart of net revenue over time
Finance export — Finance → Overview → ⬇ Export CSV
Revenue attributed by check-in date. Cleaning fees by check-out date.

📊 Google Sheets Sync
Two-way sync with a Google Sheet — push your data up or pull changes back down. Uses a free Apps Script web app as the bridge.
Not connected
Paste the Web App URL from your deployed Apps Script (see setup guide below).
Syncing…
📋 Setup Guide (click to expand)
  1. Create a new Google Sheet (sheets.google.com → Blank)
  2. Open Extensions → Apps Script
  3. Delete any existing code in the editor
  4. Open bloomstone-sheets.gs (in the same folder as this app) and paste its contents into the Script Editor
  5. Click Save (💾 icon or Ctrl+S)
  6. Click Run → setupSheet (first time only — approve permissions)
  7. Click Deploy → New Deployment
  8. Set Type: Web App · Execute as: Me · Who has access: Anyone
  9. Click Deploy and copy the Web App URL
  10. Paste the URL above and click Connect
⚠ Privacy note: Set access to "Anyone with Google Account" if you want to restrict access to signed-in Google users only.
