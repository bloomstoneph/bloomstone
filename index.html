📋 Setup Guide (click to expand)
  1. Create a new Google Sheet (sheets.google.com → Blank)
  2. Open Extensions → Apps Script
  3. Delete any existing code in the editor
  4. Open bloomstone-sheets.gs (in the same folder as this app) and paste its contents into the Script Editor
  5. Click Save (💾 icon or Ctrl+S)
  6. Click Run → setupSheet (first time only — approve permissions)
  7. Click Deploy → New Deployment
  8. Set Type: Web App · Execute as: Me · Who has access: Anyone
  9. Click Deploy and copy the Web App URL
  10. Paste the URL above and click Connect
⚠ Privacy note: Set access to "Anyone with Google Account" if you want to restrict access to signed-in Google users only.
